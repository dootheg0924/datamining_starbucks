# PU 신뢰성 평가 로그 (4기둥)

> 생성 2026-06-16 16:25:49

## 1. PU-valid 성능

- Lee&Liu PU-score: gate=1.047, max=1.165(thr=0.383)
- held-out 스벅 recall: recall@5%=0.144, recall@10%=0.238, recall@20%=0.417, recall@30%=0.548

## 2. 통계적 안정성

- PU-AUC = 0.6776, **95% CI [0.6591, 0.6976]** (bootstrap 1000)
- 모델간 점수 순위일치 Spearman: 평균 0.874, 최소 0.772 (GBM, RandomForest, XGBoost, LogReg, SVM(RBF))
  → 서로 다른 분류기가 사실상 같은 순위 = 결과가 특정 모델 산물이 아님

## 3. 외적 타당성 (OOT 신규스벅)

- recall@top-k%: top5%=0.50, top10%=0.62, top20%=0.88, top30%=0.88, top50%=0.88
  → 모델 상위 k% 안에 신규 입점 지역이 얼마나 덮이나(미지의 진실 회수)

## 4. 가정·보정

- Elkan c (E[score|라벨양성]) = 0.567, 추정 진짜 양성비율 π≈0.054 (SCAR 근사)
- PU 신뢰도곡선 단조성 ρ = 0.976 (1에 가까울수록 score↑→스벅률↑ 일관)
  *주의: 진짜 음성 부재로 절대 calibration 아님 — 순위 신뢰도(랭킹 의미보존) 확인용*

## 종합

PU에 맞는 지표(PU-score)+안정성(CI·모델일치)+외적타당(OOT)+가정정합(c·단조성)으로 신뢰 삼각형 구축. 단일 AUC가 아니라 다지표로 'PU 모델이 신뢰 가능'함을 입증.
